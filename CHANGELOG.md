# Changelog

All notable changes to the Roam Research Progress Bar Component will be documented in this file.

## [17.0.0] - 2025-04-12

### Added
- Make percentage display optional in progress bar

### Fixed
- Fix smooth hover transitions

### Changed
- Code cleanup and refactoring
- Remove embed functions (simplified architecture)
- Improved logging

## [16.0.0] - 2025-03-18

### Changed
- Simplified regex patterns for better performance
- Improved embed detection to find more embedded blocks
- Moved to `/pull` queries only for better efficiency

### Fixed
- Fixed styling issues
- Fixed inline formatting

## [15.0.0] - 2025-03-10

### Added
- Settings menu accessible via hover gear icon
- Optional embed counting feature
- Configurable display options through UI

## [14.0.0] - 2025-02-12

### Added
- Custom text labels for status display
- Ability to change "Done" label to custom text

### Fixed
- Fixed code block update issues
- Fixed size calculations

## [13.0.0] - 2025-01-08

### Fixed
- Re-added code warning functionality
- Improved setup/teardown processes
- Fixed bad UID renaming issues
- Fixed block creation errors

## [12.0.0] - 2024-12-30

### Added
- Circle/radial visualization option for progress bar

### Changed
- Increased use of CSS variables for better customization

## [11.0.0] - 2023-04-23

### Added
- Toggle for strikethrough CSS styling
- External file loading support

### Changed
- Generalized codebase for better maintainability
- Renamed files for consistency
- Improved documentation

### Fixed
- Fixed code UID setting issues

## [1.0.0] - 2023-01-12

### Added
- Initial release
- Basic progress bar functionality
- CSS styling support
- Progress bar creation on page load
- TODO/DONE counting within blocks and children
